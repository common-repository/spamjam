SpamJam stops comment spam and registration spam so you can sit back and enjoy your free time.

== Description ==

SpamJam is an antispam that protects your comment and registration forms from being abused. It can prevent 99% of all automated spam requests, transparently for end-users.

As a result, you can enjoy a bloat-free database and finalyy enjoy your free time instead of spending hours dealing with spam entries.

SpamJam has also been created to be stealth, and totally invisible for legitimate users.

== Installation ==

1. Install the plugin.
2. Activate the plugin through the 'Plugins' menu in WordPress.
